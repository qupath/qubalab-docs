QuBaLab documentation
=====================

QuBaLab is a Python package for exploring quantitative bioimage analysis... *especially* (but not exclusively) in 
combination with `QuPath <https://qupath.github.io/>`_.

Take a look at the `GitHub page <https://github.com/qupath/qubalab>`_ of the project for more details (such as more information on the project and how to install it).

This website contains notebooks that show how to use the QuBaLab package, as well as the API documentation that provides details on every features of the package.
